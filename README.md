# 📋 Name Collector

A simple and user-friendly web app for collecting and displaying names in a list.

---

## 🚀 Features

- ✅ Add names to the list using the **"Add name"** button
- 🧹 Clear all names instantly with the **"Clear All"** button
- 🔢 **Automatically counts** and displays the total number of names added
- ⚠️ Shows an **error message** when the input is empty
- 🎨 Clean UI with smooth **hover effects**
- 🔵 Each name appears as its own **individual card**

---

## 🛠️ Technologies

| Technology           | Purpose               |
| -------------------- | --------------------- |
| HTML5                | Page structure        |
| CSS3                 | Styling & animations  |
| JavaScript (Vanilla) | Dynamic functionality |

---

## ⚙️ Getting Started

1. Clone the repository:

```bash
git clone https://github.com/Azizbek432/Name-Collector.git
```

2. Open `index.html` in your browser — done!

> No additional libraries or installation required.

---

## 🎮 How to Use

1. Type a name into the **input field**
2. Click **"Add name"** or press `Enter`
3. The name is added to the list and the **total count** updates
4. Click **"Clear All"** to remove all names at once
